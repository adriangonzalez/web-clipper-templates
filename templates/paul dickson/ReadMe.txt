To install these web clipper templates:

1) Open the Web Clipper in your web browser
2) Import the .JSON files under New Template
3) Add the .css files to your .obsidian/snippets folder
4) Ensure .css snippets are enabled in Obsidian > Appearance
5) Customise to match your YAML properties
6) Set up your AI Interpreter
